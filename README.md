# Silicate-v3-macro-merger
slc3 parser used: https://github.com/silicate-bot
windows only. this program allows you to merge 1p and 2p macro in a single macro.
to use it, select 2 .slc macros, and the program will automatically merge the macros into a single one.
if you find any problems or have any suggestions, please create an issue here on github.
